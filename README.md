# hashtool


### 安装

```python
python setup.py install
```

### 使用方式
```bash
$hash -h
usage: hash [-h] [-t] file method

hash file or text with different method

positional arguments:
  file        target file
  method      the following methods are availalbe: md5, sha1, sha224,
              sha256, sha384, sha3_224, sha3_256, sha3_384, sha3_512, sha512

optional arguments:
  -h, --help  show this help message and exit
  -t, --text  hash a text instead a file
```
